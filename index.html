Parameters

The first three columns must be the names of the columns of your data! These will be entered as your vertex label in your ternary plot. If you have already entered lines, the order of the columns must be the same. For the values you can use either decimals (0.2, 0.3, 0.5) or percentages (20, 30, 50). Mixing is possible between rows.You can't mix 'm in one line though! color: hexadecimal color code like #dcedff, or just named colors text see this page for supported named colors.

Line specific
Keywords for lines are:
shape: Allowed shapes: circle (default), cross, diamond, triangle-down, triangle-up Default is circle
Title: displayed for hovering over the point with cursor, only the first row of line needs a title
Color: Only the first row/point of a line needs a color. linestyle: svg dasharrays, use 'none' for a solid lide

Lines are separated by a dash '-' or multiple dashes '-----'
