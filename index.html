Parameters

The values in the text area should be separated by commas. The first row has to contain your column names and these have to be the first three values. These will be entered as vertex labels in the ternary plot. The labels are placed in the following order: First column: top, second column: left, third column: right. Note that you have to keep the order of the your three column names consistent between points and lines.

For the values you can use either decimals 0.2, 0.3, 0.5 or percentages 20, 30, 50. Mixing these formats between rows is possible. You can't mix 'm in one line though! Also be sure to use a dot as a decimal separator.

Lines and areas are seperated by one or multiple dashes, like '-' or '------'. Only the first row of a a line or area style parameters area read. The optional parameters on following rows won't be read.

Optional Keywords

Keyword Values
Title Text (without commas and dashed). This will appended as an SVG title and will appear when hovering over the point or line.
Color (or colour) A hexadecimal color value like '#dcedff', or a named color like 'blue' or 'orangered'
Opacity Value between 1 and 0
Point only: Shape Circle, cross, diamond, triangle-down, or triangle-up
Line only: Linestyle An SVG stroke-dasharray. A list of white space separated length's and percentage's that specify the lengths of alternating dashes and gaps. The values should be separated by spaces not commas! For example: '5 15 3'. Use 'none' for a solid line.
Line only: Strokewidth The width of the line in pixels. Be warned; anything above 7 will look too chunky. Default is 4 px.
