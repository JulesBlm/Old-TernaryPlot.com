Parameters

The values in the text area should be separated by commas. The first row has to contain your column names and these have the first three values. These will be entered as vertex labels in the ternary plot. The labels are placed: First column: top, second column: left, third column: right. Note that you have to keep the order of the your three column names consistent between points and lines.

For the values you can use either decimals (0.2, 0.3, 0.5) or percentages (20, 30, 50). Mixing these formats between rows is possible. You can't mix 'm in one line though!

Optional Keywords

Keyword Values
Title Text (without commas)
Color (or colour) A hexadecimal color value like '#dcedff', or a named color like 'blue' or 'orangered'
Point only: Shape Circle, cross, diamond, triangle-down, or triangle-up
Line only: Linestyle An SVG stroke-dasharray. The values should be separated by spaces not commas! For example: '5 15 3'. Use 'none' for a solid line.
